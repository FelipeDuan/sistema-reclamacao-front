# Sistema de Reclamações — Frontend

Este é o frontend do **Sistema de Reclamações**, desenvolvido em React com TypeScript, utilizando Tailwind CSS e Vite como bundler. O sistema foi proposto como um teste técnico para uma vaga de estágio na ETIPI e teve como objetivo complementar a API backend, oferecendo uma interface moderna, intuitiva e responsiva.

## 🚀 Funcionalidades

- Cadastro de usuários
- Login (com autenticação via JWT)
- Registro de novas reclamações
- Visualização das reclamações cadastradas
- Edição e exclusão de reclamações (enquanto pendentes)
- Atualização de status para "Respondido"
- Interface limpa e amigável, focada na experiência do usuário

## 💻 Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Fetch API** (para comunicação com o backend)

## 🎯 Objetivos

O projeto buscou entregar uma experiência fluida e intuitiva ao usuário final. Além de consolidar conhecimentos técnicos, foi fundamental para meu desenvolvimento pessoal, permitindo aplicar boas práticas de front-end, integração com API e autenticação segura.
